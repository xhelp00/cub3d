### Controls
| Key   | Action          |   
|-------|-----------------|
|   W   | move forward    |
|   S   | move back       |
|   A   | move left       |
|   D   | move right      |
|   Q   | rotate left     |
|   E   | rotate right    |
| Space | jump            |
| Shift | sprint          |
| Ctrl  | crouch          |
| F8    | hide / show HUD |
| LMB   | shooting        |
| RMB   | action          |
| IDDQD | GOD MODE        |
-------------------------------------------------
